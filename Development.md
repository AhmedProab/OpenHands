### Model Integration Types
1. **Single Agent Mode**
   - One model for all tasks
   - Default: qwen2.5-coder:32b

2. **Multi Agent Mode**
   - Task-specific model distribution
   - Supported models:
     - llama3.3 (42 GB)
     - gemma:7b (5.0 GB)
     - gemma:2b (1.7 GB)

3. **Hybrid Mode**
   - Smart task delegation
   - Optimized resource usage
